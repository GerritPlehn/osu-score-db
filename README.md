# Bun Template

## Included things

- Bun (runtime)
- Biome (formatter & linter)
- @t3oss/env (type-safe env variables)
- lefthook (Git Hooks)
- VSCode configs (format on save, debug config, recommended extensions)

## Usage

1. Fork/use template
2. `bun install` install dependencies
3. `bunx lefthook install` register git hooks
